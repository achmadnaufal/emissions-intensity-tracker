# Changelog

## [1.5.0] - 2026-03-03

### Added
- Emissions Scope Calculator for GHG quantification
- Scope 1 (diesel/fuel) emissions calculation
- Scope 2 (electricity) emissions calculation
- Scope 3 (shipping/transportation) emissions calculation
- Configurable emission factors (currently coal mining defaults)
- Unit tests for all scope calculations (4 test cases)
- Sample emissions data across multiple mines
- Comprehensive edge case handling

### Features
- Built-in emission factors for coal operations
- Support for multiple mines and time periods
